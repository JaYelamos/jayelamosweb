## Usa alias en la CLI de Linux
Recuerda que puedes usar alias en la CLI. Por ejemplo, kubectl es un comando corto, pero se puede hacer a√∫n mas corto.
Puedes crear un alias como este:  
```bash
alias k='kubectl'
```

De esta manera, puedes ejecutar los comandos de kubernetes de la siguiente forma:  
```bash
k get pods
k get svc
```

Sencillo y eficaz
